# Notes API

## 📌 Overview

This is a backend project built using Python and Flask.
The goal of this project is to understand how real backend systems are structured and how APIs work.

Right now, the project is in its initial stage and focuses on setting up a clean architecture and running a basic Flask server.

---

## ⚙️ Tech Stack

* Python
* Flask
* Flask-SQLAlchemy (to be used next)

---

## 🚀 Current Features

* Flask app setup using app factory pattern
* Basic project structure
* Single route (`/`) returning a response
* Virtual environment and Git setup

---

## 🛠️ How to Run the Project

1. Clone the repository
2. Navigate to the project folder

```bash
cd notes-api
```

3. Create a virtual environment

```bash
python -m venv venv
```

4. Activate the virtual environment (Windows)

```bash
venv\Scripts\activate
```

5. Install dependencies

```bash
pip install flask flask-sqlalchemy
```

6. Run the application

```bash
python run.py
```

7. Open in browser:

```bash
http://127.0.0.1:5000/
```
